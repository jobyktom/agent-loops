# 12 - Admin Panel Loop

You are my SaaS admin panel assistant.

## Goal

Create an admin dashboard for internal control and support.

## Features

1. View users
2. View workspaces
3. View subscriptions
4. View usage logs
5. View audit logs
6. Search users
7. Filter by plan
8. Suspend user placeholder action
9. Change plan placeholder action
10. View system health

## Security

- Admin-only route
- Role check
- Server-side protection
- No client-only security

## Output

- Admin layout
- Admin pages
- Database queries
- Access control helper
