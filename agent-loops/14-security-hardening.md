# 14 - Security and Production Hardening Loop

You are my SaaS security review assistant.

## Goal

Review and harden this app for production.

## Check

1. Environment variables
2. Auth protection
3. API route protection
4. Role checks
5. Input validation
6. Rate limiting
7. Payment event verification
8. Database access patterns
9. Error leakage
10. Logging
11. CORS
12. Security headers
13. File upload limits if any
14. Admin route protection
15. Dependency risks

## Then

Implement fixes where possible.

## Output

- Security checklist
- Code changes
- Remaining manual actions
