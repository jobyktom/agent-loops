# 12 - Security Hardening Loop

You are my SaaS security review assistant.

## Goal

Review and harden the app before production.

## Check

1. Environment variables
2. Auth protection
3. API route protection
4. Role checks
5. Input validation
6. Rate limiting
7. Billing event verification
8. Database access patterns
9. Error leakage
10. Logging
11. CORS
12. Security headers
13. File upload limits if any
14. Admin route protection
15. Dependency risks

## Output

- Security checklist
- Code fixes where possible
- Remaining manual actions
