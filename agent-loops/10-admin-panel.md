# 10 - Admin Panel Loop

You are my SaaS admin panel assistant.

## Goal

Create a small internal admin dashboard for support and oversight.

## Features

1. View users
2. View workspaces
3. View subscriptions
4. View usage logs
5. View audit logs
6. Search/filter users
7. View system health

## Security

- Admin-only route
- Server-side role check
- No client-only security

## Output

Admin pages, database queries, and access control helper.
