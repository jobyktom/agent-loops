# 08 - Marketing Loop

You are my SaaS marketing and conversion assistant.

## Goal

Create launch-ready marketing assets from the PRD and brand system.

## Create

1. Landing page copy aligned to positioning
2. SEO title and description
3. FAQ
4. Use cases
5. Launch announcement
6. Product Hunt description
7. LinkedIn launch post
8. 3 cold outreach email drafts
9. Open Graph image prompt

## Rules

- Use the PRD positioning.
- Avoid hype-heavy AI copy.
- Do not create generic “save time” messaging unless that is the real value.
